CSS Files and Structures

This template is built on Bootstrap 4.3.x framework which can be found in bootstrap/css folder

All CSS (except Bootstrap) files can be found in the templateâ€™s css folder. Here's the list of the files used within the template and a short descriptions of their function:

  • font-icons.css - all icon fonts in this file,
  • animate.min.css - CSS animations,
  • main.css - reset styles, fonts , font icons, etc.,
  • plugin.css - plugins (javascript, jquery, etc.,) styles
  • style.css - template styles
  • your-style.css - for your own style

Javascript and jQuery

This template imports several Javascript libraries and files, i have put a link to original authors on the file names (ended with .js) where you can find more documentation on each of them:

  • Main jQuery Library - jquery.min.js
  • jQuery Migrate - jquery-migrate-1.4.1.min.js
  • Bootstrap's JavaScript components - bootstrap.min.js
  • jQuery Waypoints is the easiest way to trigger a function when you scroll to an element. - jquery.waypoints.min.js
  • jQuery Easing Plugin is a jQuery plugin from GSGD to give advanced easing options. - jquery.easing.1.3.js
  • Bootstrap Rating is a jQuery plugin that creates a rating control that uses Bootstrap glyphicons for rating symbols. - bootstrap-rating.js
  • jQuery InfoBox Plugin a plugin to customize google map infowindow - infobox.js
  • jQuery Slick Plugin is fully responsive carousel - slick.min.js
  • Ion.RangeSlider is a jQuery-plugin Easy, flexible and responsive range slider with skin support - ion.rangeSlider.min.js
  • Chosen - a jQuery Plugin by Harvest to Tame Unwieldy Select Boxes. - chosen.jquery.min.js
  • Bootstrap Touchspin - is a mobile and touch friendly input spinner component for Bootstrap. - jquery.bootstrap-touchspin.min.js
  • Air Datepicker - is a lightweight cross-browser jQuery datepicker. - air-datepicker.min.js
  • Sticky-Kit - is a jQuery plugin for making smart sticky elements. - sticky-kit.min.js
  • Moment - parse, validate, manipulate, and display dates in JavaScript. - moment.min.js
  • jQuery Mask Plugin - A plugin to make masks on form fields. - jquery.mask.min.js
  • jQuery Countimator - is an animated counter jQuery plugin. - jquery.countimator.js & jquery.countimator.wheel.js
  • Bootstrap Validator - is an user-friendly HTML5 form validation jQuery plugin for Bootstrap. - validator.min.js
  • Bootstrap ScrollSpy - is for automatically updating nav targets based on scroll position. Scroll the area below the navbar and watch the active class change. - affix.js
  • custom-core.js - core customized to jQuery, needed for every pages
  • custom-contact-page.js - customized to jQuery, used only for contact page
  • custom-detail-map.js - customized to jQuery for google map in only detail pages
  • custom-map-route.js - customized to jQuery for google map route in a detail page
  • custom-multiply-sticky.js - customized to jQuery sticky navigator in detailed pages
